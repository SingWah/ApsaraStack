# ApsaraStack 代码分析

分析 阿里云专有云ApsaraStack代码库，生成交互式 HTML 分层架构图。支持 Python、Java、Go 多语言，自动识别函数调用链、类依赖关系、HTTP/API 接口和外部服务依赖，将代码结构可视化为入口层→接口层→控制层→服务层→数据层→模型层→基础设施层的七层架构图。


> **说明：** 本插件基于静态代码分析，生成的架构图需要结合实际运行逻辑进行校验。

## 目标角色

- **运维工程师** — 快速理解各子系统代码架构，定位问题根因
- **平台开发工程师** — 梳理模块间调用关系，评估变更影响范围
- **技术架构师** — 分析代码分层合理性，识别架构改进点

## 快速命令

| 命令 | 说明 |
|------|------|
| `/代码调用分析` | 选择代码目录，分析并生成 HTML 架构图 |

## 技能

| 技能 | 说明 |
|------|------|
| 代码调用分析 | 对指定目录进行多维度静态分析（函数调用、类继承、API 接口、外部依赖），生成交互式分层架构图 HTML 文件，支持在浏览器中直接打开和复制内容 |
